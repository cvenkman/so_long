## So_Long

Function that returns a line ending with a newline, read from a file descriptor

### Compiling

* To compile
	- `make`
* To remove objects:
	- `make clean`
* To remove objects and binary file (program):
	- `make fclean`
* To re-compile:
	- `make re`

### Executing

`./so_long maps/map2.ber`
